# 試験タイマー

マークシート式の試験・模試などを **一定ペースで解く** ための、ブラウザ完結のタイマーアプリ。
「全○問を△分で、1問あたり□分のペースで進める」形式の試験なら、設定を変えるだけで何にでも使えます。
（既定値は 50問・120分・1問2分・開始13:00。設定で自由に変更できます）

## 特長

- **現在の問題番号**「第 N / 総数 問」を大きく表示
- **アナログ時計**を中央に表示。針はタイマー（経過時間）と連動し、稼働中は進み一時停止で止まる
  - 開始時刻を設定可能（既定 13:00）。腕時計しか持ち込めない本番環境に近い形で練習できる
- **この問題の残り時間**を目安時間からカウントダウン（残り少→黄、超過→赤点滅＋ビープ）
- **全体の残り時間**と**ペース**（予定より ＋/− 何分何秒）を可視化
- **目安時間が経過したら自動で次の問題へ**進む（設定でオン/オフ）
- **途中の問題から開始**できる（問題数表示をクリックして選択。選んだ問題以降の持ち時間に残り時間を自動調整）
- 単一 HTML・ビルド不要・1画面に収まる（スクロール無し）・スマートフォン対応

## 設定（⚙）

すべて localStorage に保存されます。

| 項目 | 既定 | 説明 |
|---|---|---|
| 問題数 | 50 | 全問題の数 |
| 制限時間 | 120 分 | 全体の制限時間 |
| 1問の目安 | 120 秒 | 1問あたりの目安時間（ペース判定の基準） |
| 開始時刻 | 13:00 | アナログ時計の始まりの時刻 |
| 自動で次の問題へ | オン | 目安経過で自動的に次問へ進むか |
| 終了音 | オン | 目安経過・終了時にビープを鳴らすか |

## 使い方

`index.html` をブラウザで開くだけ（インターネット接続不要）。

| 操作 | 動作 |
|---|---|
| スタート / `Space` | スタート / 一時停止 / 再開 |
| 次の問題へ / `→` / `Enter` | 次の問題へ進む |
| 前の問題 / `←` | 前の問題へ戻る |
| 「第 N / 総数 問」をクリック | 開始する問題を選択 |
